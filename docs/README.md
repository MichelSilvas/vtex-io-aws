## Como vizualizar
Apos clonar o repositório, em um terminal entre em sua conta Vtex
```bash
vtex login {conta}
```
Entre (ou crie) em um workspace de desevolvimento
```bash
vtex use examplename
```
Inicie uma sessão de desenvolvimento
```bash
vtex link
```
Para visualizar no navegador use
```bash
vtex browse
```
